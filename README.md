# Prity's Beauty Parlour Website

This is the official landing page design for Prity's Beauty Parlour (Katihar). It is built using HTML and Tailwind CSS.

## Features
- **Luxury Aesthetic:** Rose Gold and Maroon color scheme.
- **Mobile First:** Looks great on phones.
- **Direct Booking:** WhatsApp and Call buttons are pre-linked.
- **Location:** Embedded Google Map for Durgasthan, Katihar.

## How to Edit
1. Open `index.html` in any text editor (Notepad, VS Code).
2. **Images:** Search for "unsplash" links in the code and replace them with links to your own bridal photos.
3. **Services:** You can change the text inside the "Services" section to update your offerings.

## How to Deploy (Go Online for Free)
1. Create a GitHub account.
2. Create a new repository named `pritys-beauty-parlour`.
3. Upload `index.html`.
4. Go to **Settings > Pages**.
5. Under "Branch", select `main` and click Save.
6. Your website will be live at `yourusername.github.io/pritys-beauty-parlour`.
